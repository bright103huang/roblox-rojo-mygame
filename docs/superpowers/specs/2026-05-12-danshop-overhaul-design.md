# 仙丹阁改造设计文档

## 背景
仙丹阁场景目前只有一片空场地 + 隐形触碰区域，没有 NPC、没有柜台展示、玩家进入不知道该做什么。本次改造从场景视觉和交互玩法两个维度重做仙丹阁。

## 场景布局

### 尺寸（2D 横版）
- 地面：80 宽 × 8 深（原为 40 宽，翻倍显大气）
- 后墙：12 格高（原为 6 格）
- Z 轴：后层 Z=-4，交互层 Z=0，前层 Z=4

### 空间布局
```
Z=-4 后层装饰：
  ┌────────────────────────────────────────────┐
  │  整面药柜墙，6 个展示格，各放发光颜色瓶    │
  │  [丹][药][灵][仙][宝][神]                  │
  │  ═══════════════════════════════════ 柜台  │
  │         [掌柜 NPC]                         │
  │                                            │
  │   ← 玩家从左侧进入 (Z=0)                   │
  │                                            │
  │  灯笼  盆栽  门槛石                        │
  └────────────────────────────────────────────┘
```

### 视觉元素
- **药柜墙**：后墙整面展示柜，每格 4 宽，隔板用薄板拼接
- **柜台**：深色木纹长台（约 24 宽），台面放 3-4 瓶样品丹（小号彩色圆柱）
- **牌匾**：柜台上方悬挂，「仙丹阁」大字
- **前层**：入口两侧灯笼（红黄配色）、盆栽（绿+棕）、门槛台阶
- **地面**：暖色地砖（浅棕/米黄间隔，可加网格线）

## NPC 掌柜

### 造型（Roblox Part 拼接）
总计约 16 个零件，通过 WeldConstraint 连接到透明 Root 上（参照 BeastNPC.buildAnimalShape 模式）。

| 部位 | Shape | 颜色 | 备注 |
|------|-------|------|------|
| 袍身上段 | Cylinder | 深红(Dark red) | 主体 |
| 袍子下摆 | Cylinder 加宽 | 深红 | 垂感 |
| 头部 | Sphere | 肤色(Warm skin) | |
| 帽子 | Sphere+Ring | 黑色 | 中式员外帽 |
| 眼睛×2 | Sphere | 白色(Neon) | 发光眼白 |
| 瞳孔×2 | Sphere | 黑色 | 嵌在眼白上 |
| 眉毛×2 | Wedge | 深灰 | 细长 |
| 胡子 | Wedge | 深灰 | 下巴下 |
| 双臂×2 | Cylinder | 深红 | 袖口 |
| 双手×2 | Sphere | 肤色 | 袖口外露 |
| 腰带 | Block | 金色(Bright yellow) | 腰间 |

### NPC 动画
- **呼吸浮动**：Root 每 2s 上下浮动 0.3 格（Tween 循环）
- **头部微转**：头部零件绕 Y 轴 ±15° 摆动
- **招呼手势**：右臂每 6-10s 抬一次（模拟招呼顾客）
- **注视跟随**：检测最近玩家，头部朝玩家方向微偏

### 交互
- 触碰 NPC 任意零件 → 触发 `ShopEvent:FireServer("Pick:Shop")`
- NPC 头顶 BillboardGui 显示「掌柜」
- 交互区域以 NPC 为中心，半径 4 格

## 砍价系统

### 触发流程
1. 触碰 NPC → 服务端返回商品数据 → 打开 ShopUI
2. 在 ShopUI 中选择商品 → 点击「砍价」按钮
3. 弹出对话弹窗：老板说一句话，玩家 3 选 1
4. 选到让老板开心的选项 → 该商品 8 折
5. 选错 → 原价

### 规则
- 每次打开 ShopUI，每件商品可砍价 1 次
- 关闭面板重新打开可再次尝试
- 无好感积累、无永久折扣、无惩罚
- 折扣仅影响当前这笔交易

### 评判原则
老板性格：精明、爱面子、喜欢被捧、讨厌威胁和哭穷。

- **让老板开心**（8折）：奉承、幽默、互惠提议、有趣的话题
- **老板不开心**（原价）：威胁、贬低、哭穷、抬杠

  ✅ "老板好眼力！所以需要丹药补补"

  ❌ "你怎么知道"

  ✅ "难怪灵气这么足，值这个价！"

## 涉及文件

| 文件 | 改动内容 |
|------|---------|
| `SceneSetup.server.lua` | 重写 `setupShopScene()`：80 地面、药柜、柜台、牌匾、NPC 掌柜生成、灯笼盆栽、触碰绑定 |
| `Systems/ShopkeeperNPC.lua` **(新文件)** | NPC 零件拼接 + 动画循环 + 交互区域绑定 |
| `ShopService.server.lua` | 新增砍价判定 `HandleBargain(player, itemKey, choiceIndex)` → 返回 `{ success, discount }`；对话题库 |
| `ShopUI.local.lua` | 新增「砍价」按钮、砍价对话弹窗 UI（老板对话 + 3 选项）、结果反馈 |
| `DataManager.lua` | 无改动（砍价不存数据） |

## 未改动
- `DanConfig.lua` — 丹药品类和价格不动
- `Config/init.lua` — 无需新增配置
- `TaskClient.local.luau` — 不涉及任务系统
- `SceneChoiceUI.lua` — 仙丹阁在场景选择中不变

## 验证方法
1. 进入仙丹阁 → 看到宽敞大厅、药柜、柜台、牌匾、掌柜 NPC 在柜后
2. NPC 有呼吸浮动、头部微转等动画
3. 触碰 NPC → 打开 ShopUI
4. 选商品 → 点「砍价」→ 弹老板对话 + 3 选项
5. 选让老板开心的选项 → 提示"老板很开心！8 折！"
6. 选其他选项 → 提示"老板不高兴，原价购买"
7. 购买时按正确价格扣仙晶
8. 关面板重新进，砍价重置
